# applybn

applybn — это многоцелевой фреймворк с открытым исходным кодом, основанный на байесовских сетях и каузальных моделях.
Основная идея заключается в реализации основных функций анализа данных на основе понятных и интерпретируемых алгоритмов байесовских сетей и каузальных моделей.
